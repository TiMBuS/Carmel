# NAME

Menlo - A CPAN client

# DESCRIPTION

Menlo is a code name for _cpanm 2.0_, developed with the goal to
replace [cpanm](https://metacpan.org/pod/cpanm) and its backend with a more flexible, extensible and
easier to use APIs.
